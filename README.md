EcoLife (Expo + TypeScript)

Мобільний застосунок для підтримки екологічного способу життя. Застосунок призначений для допомоги користувачам у пошуку пунктів сортування відходів, отриманні інформації про можливості утилізації та формуванні екологічних звичок у повсякденному житті.

Основні можливості застосунку включають відображення карти пунктів сортування відходів, навчальний розділ «PandaTeach», який містить освітні матеріали екологічної тематики, інтеграцію із штучним інтелектом для отримання додаткової інформації, а також інші функції.

Для запуску проєкту необхідно встановити залежності за допомогою команди npm install, після чого запустити застосунок командою npx expo start -c. Далі слід відкрити застосунок у Expo Go на пристрої Android та відсканувати QR-код у режимі LAN.

У проєкті використано такі технології: React Native (Expo), TypeScript та Supabase для роботи з базою даних.

Застосунок спрямований на підвищення екологічної свідомості користувачів та полегшення доступу до інформації про сортування відходів.
